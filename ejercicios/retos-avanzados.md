# Retos Avanzados: Backtracking y Optimización

Problemas complejos que requieren exploración de estados o memorización.

## 1. Generar Paréntesis

Dado `n`, genera todas las combinaciones de paréntesis bien formados.
Ejemplo `n=3`: `["((()))", "(()())", "(())()", "()(())", "()()()"]`

## 2. Permutaciones

Genera todas las permutaciones posibles de un array de números únicos.
`[1, 2, 3]` -> `[[1,2,3], [1,3,2], [2,1,3], ...]`

## 3. Salto del Caballo (Knight's Tour)

Dado un tablero NxN, encuentra una secuencia de movimientos de un caballo de ajedrez que visite cada casilla exactamente una vez.

## 4. Fibonacci con Memoización

Optimiza la función de Fibonacci para que sea O(n) usando un objeto o Map para guardar resultados previos.
