# Ejercicios Nivel 1: Conceptos Básicos

Estos ejercicios están diseñados para familiarizarte con la estructura básica de una función recursiva: caso base y llamada recursiva.

## 1. Conteo Regresivo

Escribe una función recursiva que imprima los números desde `n` hasta `1`.

## 2. Suma de Dígitos

Dado un número entero positivo (ej: 123), devuelve la suma de sus dígitos (1+2+3 = 6).
_Pista: Usa `% 10` para obtener el último dígito y `/ 10` para quitarlo._

## 3. Invertir String

Invierte una cadena de texto usando recursividad.
Ejemplo: `"hola"` -> `"aloh"`.

## 4. Potencia

Calcula `base` elevado a la `exponente` de forma recursiva.
`potencia(2, 3) = 8`
